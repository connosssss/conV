chiika is an app made to let users practice real world conversations using Gemini live. You can put in a target language and then choose a target scenario to simulate to practice a conversation. Because it's running real time, you can also ask additional questions whenever you want while also changing topics whenever you would like.

Kaggle Link: https://www.kaggle.com/competitions/ucla-gemini-api-hackathon/writeups/chiika-real-time-conversations
Deployment Link: https://chiika.vercel.app/
